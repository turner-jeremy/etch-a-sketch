Instructions

Click and drag mouse within etching area to draw.


Grid Size

Adjust number of squares on each axis, ranging from 2 to 100.

Changing grid size will automatically clear etching area!

Colour

Changes colour of sketching tool.

Random will change colour each time you select this option.

Confetti will change colour each time you move to a new square.

Decay Time

Number of seconds that it takes for sketch to disappear.

Moving slider all the way to the left turns off fade effect.

Moving slider all the way to the right will lock sketch from fading.

Clear

Clears all sketches currently on the etching space.
